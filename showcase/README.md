## Privacy Note

**IMPORTANT:** Screenshots and demos should show:
- ✅ User interface
- ✅ Query results
- ✅ Performance metrics
- ✅ System architecture (high-level)

**NEVER show:**
- ❌ Source code of proprietary components
- ❌ Model weights or configurations
- ❌ Internal API endpoints
- ❌ Database schemas
- ❌ Sensitive document content
